## Usage

1) Download or Clone the repository

```bash
$ git clone https://github.com/akpw/VisualBinaryTrees
```

3) Open the playground in Xcode

```bash
$ open "VisualBinaryTrees.playground"
```
# Requirements

* Xcode 8
* iOS 10
